# CISK rapportmal i LaTeX
Dette er en rapportmal for prosjekter, rapporter og andre akademiske tekster under studietiden ved CISK.

Det er anbefalt å laste ned LaTeX-extensions i Visual Studio Code. "LaTeX","LaTeX Workshop" og "LaTeX Utilities".
Samt at det kreves MikTex-console, og Strawberry Perl for at det skal kunne kjøre og kompilere for Windows brukere. Jeg er ingen LaTeX ekspert, men last ned disse og installer packages i MikTex-console. Strawberry Perl er plug-and-play.

For Linux brukere: sudo pacman -S biber, sudo pacman -S texlive-langextra for å installere siteringspakken, og norsk språk. Generelt så er det bare å følge feilmeldingen og installere det den ber om, men disse to er pakker jeg måtte ha.

Se på YT for annen assistanse :)
